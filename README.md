## ⚡ 1-Minute Upgrade

Start with these basics:

- `htop` → launch interactive monitor  
- `F3` → search processes  
- `F5` → tree view  
- `F9` → kill process  

💡 Once you use `htop`, you won’t go back to `top`.

## 🚀 Quick Install (Copy-Paste Ready)

```bash
# Debian/Ubuntu
sudo apt install htop -y

# RHEL/CentOS
sudo dnf install htop -y

# Arch Linux
sudo pacman -S htop
```

---

## 🧪 Basic Usage

```bash
# Start htop
htop

# Navigate using arrow keys
# Press F3 to search
# Press F5 for tree view
# Press F9 to kill a process
```
